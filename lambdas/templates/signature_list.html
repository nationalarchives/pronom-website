This page provides an archive of all previously released DROID signature files. These signature files are created by PRONOM and used by the DROID software tool to identify the exact format of a digital file.

Each download is an XML file containing a set of signatures that were released together. If you are researching digital file formats, or you are interested in how file format identification works, these files may be helpful.
