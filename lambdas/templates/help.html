Welcome to PRONOM

PRONOM is an online database that provides clear, reliable information about digital file formats.

It holds the details of over two thousand individual file formats including how to identify them. This information has been created through research by The National Archives (UK) and contributions by many archives and related organisations across the world

PRONOM is widely used as part of digital preservation activities to help organisations understand the files they hold so that they can manage their readability over time.

PRONOM data can be used in other tools, such as DROID, which is a software application developed by The National Archives (UK). DROID scans files and uses PRONOM data to identify them, helping users identify, manage, understand and preserve their digital content.

PRONOM is freely available for anyone who needs reliable information about file formats.

Using PRONOM

PRONOM contains detailed information about file formats including:

  • File format names and versions
  • File format details, background information and links to further resources
  • Technical details used for file format identification

You can search PRONOM in a few ways, including:

  • By file format name, for example pdf
  • By PRONOM Unique Identifier (PUID) for example fmt/1
  • By file extension, by preceding this with a dot, for example .pdf

Accessing PRONOM Data

PRONOM data is openly available and can be accessed in different ways.

  • Searching the website
  • Accessing the full dataset on GitHub as JSON files
  • Downloading XML files containing identification information, by appending .xml to a format page address, e.g. https://pronom.nationalarchives.gov.uk/fmt/1.xml

For more technical details, including documentation and source code, please visit the PRONOM GitHub repository

Contributing to PRONOM

We actively encourage the submission of new information to improve PRONOM. If you have information on a file format that is not currently in the database; believe you could improve an entry; or have feedback, then please do get in touch.

Currently, to contribute to PRONOM, the easiest way is to submit an issue via our PRONOM Research GitHub repository, where a member of the team can pick it up easily. We can also be contacted via email at pronom@nationalarchives.gov.uk

We are currently working on improving our contribution process, and this guidance will be updated as new options become available.

Status of this website

This is a new (beta) version of the PRONOM website, to replace the current system, which has been in use for over 20 years.

The new PRONOM is designed to:

  • Improve accessibility, usability and sustainability
  • Support open and community-driven development
  • Make it easier to access and reuse data
  • Increase transparency by using open-source tools

While the core PRONOM data remains consistent between the two systems, the new PRONOM changes how the system is delivered by:

  • Using modern, cloud-based technology
  • Making data openly available via GitHub
  • Supporting easier integration with external tools

During the Transition

  • The current PRONOM service will remain available (the website address will not change)
  • Data will continue to be released in both systems
  • The data output of both systems will continue to work on tools such as DROID

We expect to replace the current website with this new one in the near future, but it will remain available in the meantime. As soon as we have a timeline for this work we will be making announcements to the community and updating our web pages.

Feedback

The new PRONOM website is still in an early beta version, and we want to make sure to get as much feedback as possible before taking it live and using it as our main website. If you have any suggestions for improvements, please fill out our feedback form.

We will be actively contacting developers and service providers that use PRONOM data within their tools to support a smooth transition. Please get in touch if you think you might be impacted by these changes.
