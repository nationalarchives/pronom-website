PRONOM is an online database that provides clear, reliable information about digital file formats.

About this service

Archivists and others use it to understand the technical details behind files so they can keep important digital records accessible over time. A key part of PRONOM is its database of file‑format signatures - small patterns inside a file that help tools like DROID identify exactly what format it is. Knowing the precise format helps you plan how to maintain access and readability, especially as older formats become harder to open.

DROID Signature Files

These signature files are created by PRONOM and used by the DROID software tool to identify the exact format of a digital file.
